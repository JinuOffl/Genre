# Genre
Genre is a Android movie streaming app, allows users to browse and watch movies by name, cast, genre, and poster. It uses Firebase Realtime Database to store and manage movie data, ensuring seamless and fast access to movie details and video streaming. Developed with Android Studio, Java, and XML

Features :
Stream movies directly from the app,
View movie details (title, poster, cast & crew),
Firebase Realtime Database for dynamic content,
Modern UI designed with XML & Java,
Logo and UI elements created with Figma.

Language: Java, 
UI: XML, 
Backend / Cloud: Firebase Realtime Database
